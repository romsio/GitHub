avec le display flex

Le module des boîtes flexibles, aussi appelé « flexbox », a été conçu comme un modèle de disposition unidimensionnel et comme une méthode permettant de distribuer l'espace entre des objets d'une interface ainsi que de les aligner.

Code css =
display:flex;

La propriété align-items permet d'aligner les éléments le long de l'axe secondaire.

avec le display flex & l'alignement et la justification :

La propriété CSS justify-content indique la façon dont l'espace doit être réparti entre et autour des éléments selon l'axe principal d'un conteneur flexible ou selon l'axe en ligne lorsque le conteneur est une grille.

Code css =
display: flex;
align-items: center;
& justify-content: center;

avec le display flex & l'alignement et la justification & la direction :

La propriété flex-direction définit la façon dont les éléments flexibles sont placés dans un conteneur flexible : elle définit l'axe principal et la direction des éléments (normale ou inversée).

Code css =
display: flex;
align-items: center;
& justify-content: center;
flex-direction row
